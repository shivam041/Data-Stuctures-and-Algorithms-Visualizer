# Screenshots Directory

Add your project screenshots here with the following filenames:

- `overview.png` - Main interface overview showing the modern dark theme
- `quick-sort.png` - Quick Sort algorithm visualization in action
- `bubble-sort.png` - Bubble Sort algorithm visualization in action

## Recommended Screenshot Guidelines

1. **Overview Screenshot**: Capture the full interface showing both the code editor and visualization panels
2. **Algorithm Screenshots**: Show different algorithms being visualized with color-coded elements
3. **Image Format**: PNG format recommended for best quality
4. **Dimensions**: 1920x1080 or similar aspect ratio for best display on GitHub
